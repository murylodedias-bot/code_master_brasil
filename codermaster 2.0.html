O que é Linguagem de Programação?

Uma linguagem de programação é um conjunto formal de regras e sintaxe que permite aos humanos comunicar instruções a computadores. É a "língua" que usamos para criar softwares, aplicativos, sites e sistemas.


Assim como idiomas humanos (português, inglês), cada linguagem de programação tem sua própria gramática (sintaxe), vocabulário (palavras-chave) e forma de expressar ideias (paradigmas).

Python

Python é uma linguagem de programação de alto nível, criada por Guido van Rossum em 1991. É conhecida por sua sintaxe limpa e legível, próxima ao inglês natural.


Usada em ciência de dados, IA, automação, web (Django/Flask) e muito mais. É a linguagem mais popular do mundo em 2024.


Interpretada Dinâmica Multiparadigma Open Source

Java

Java é uma linguagem de programação orientada a objetos, criada pela Sun Microsystems em 1995. Seu lema: "Write once, run anywhere" (escreva uma vez, rode em qualquer lugar).


É amplamente usada em aplicativos Android, sistemas bancários, aplicações corporativas e back-end de grandes empresas.


Compilada/JVM Tipagem Forte Multiplataforma OOP
Exemplo.java
public class Exemplo { // Atributos private String nome; private int idade; // Construtor public Exemplo(String nome, int idade) { this.nome = nome; this.idade = idade; } // Método public void saudacao() { System.out.println( "Olá, " + nome ); } public static void main(String[] args) { Exemplo e = new Exemplo("Dev", 20); e.saudacao(); } }

JavaScript

JavaScript é a linguagem da web. Criada em 1995 por Brendan Eich, é a única linguagem nativamente entendida pelos navegadores. Com Node.js, também roda no servidor.


Usado em front-end (React, Vue), back-end (Node.js), apps mobile (React Native) e muito mais.


Interpretada Dinâmica Event-driven Assíncrona
script.js

C#

C# (C Sharp) é uma linguagem desenvolvida pela Microsoft em 2000. Fortemente tipada e orientada a objetos, roda na plataforma .NET.


Usada em jogos (Unity), apps Windows, desenvolvimento web (ASP.NET) e sistemas corporativos.
